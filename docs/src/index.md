# CassandraML.jl

## Introduction

## Installation
To install `CassandraML.jl`, press `]` to activate Package mode and type the following command:

```julia
pkg> add CassandraML
```

## Quick start
Assuming `CassandraML.jl` has been installed, start using the package as follows:

```julia
using CassandraML
```

See package documentation for more syntax options.
